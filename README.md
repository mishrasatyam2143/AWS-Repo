## How to Use
1. Navigate to the service folder, such as `Compute/EC2`.
2. Read the guide and follow the steps.
3. Use the provided scripts for faster implementation.

---

## Topics Covered
- Compute: EC2, Lambda, EKS  
- Storage: S3, EBS, EFS  
- Networking: VPC, Route 53, Load Balancers  
- Security: IAM, KMS, Secrets Manager  
- Database: RDS, DynamoDB  
- Monitoring: CloudWatch, CloudTrail  
- Automation: CloudFormation, Terraform
